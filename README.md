📊 Syntecxhub Project – Sales Analysis
🔍 Objective

The goal of this project is to analyze a retail sales dataset and extract meaningful business insights using Python.

📁 Dataset

The dataset contains:

Order ID

Date of purchase

Product category

Region

Revenue

⚙️ Tools & Technologies

Python

Pandas

Matplotlib

📌 Key Analysis Performed

Calculated Total Revenue

Computed Average Order Value

Identified Top Performing Products

Analyzed Region-wise Sales

Studied Monthly Sales Trends (Seasonality)

📈 Visualizations

The project includes:

Bar Chart → Top Products by Revenue

Bar Chart → Revenue by Region

Line Chart → Monthly Revenue Trend

💡 Key Insights

Some products generate significantly higher revenue than others

Sales performance varies across regions

Monthly trends indicate possible seasonality patterns
